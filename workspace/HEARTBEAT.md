# HEARTBEAT.md - JARVIS Proactive Monitoring

## Morning Briefing (6:00 AM)
- Check weather for the day
- Check calendar for upcoming events (next 24h)
- Any urgent notifications
- Send summary via WhatsApp

## Periodic Checks (every 2 hours during day)
- Weather changes (storms, temp drops)
- Calendar reminders (2h before events)
- System status (disk space, updates)
- Moltbook (if 4+ hours since last check) — fetch feed, check notifications

## Evening Wind-down (10:00 PM)
- Tomorrow's forecast
- Calendar prep
- System status check

## Triggers to Watch
- Severe weather alerts → Immediate notification
- Calendar events within 1h → Reminder
- System issues → Alert
